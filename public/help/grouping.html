Descripción de los Campos

  • Nombre:

    Este campo es para hacer referencia al nombre de blabla

  • Descripcion:

    Este campo es para tener una breve descripción de los que es blabla

  • Comuna:

    En este campo debemos seleccionar la comuna, en la cual se encuentra ubicado el blabla. para que se habilite el selector, debemos seleccionar previamente una region

  • Dirección:

    En este campo debemos completar con la dirección en la cual se encuentra ubicado el blablabla

Operaciones

  • Agregar Nuevo:

    Para agregar un nuevo blabla, primero debemos seleccionar blabla en la barra de navegación (a la izquierda).

    Veremos una pantalla que nos listará todos los blabla registrados. Presionamos en el boton "Nuevo Blabla" de color azol, en la esquina superior derecha.

    Nos cargará el formulario de ingreso. Llenamos la informacion correspondiente (Los campos obligatorios se destacan con un *).

    Damos al botón guardar.

    Luego de esto el sistema nos enviará a la página principal de Blablas y ahora nuestro blabla se mostrará en la tabla, con lo que confirmamos que está registrado en el sistema.
